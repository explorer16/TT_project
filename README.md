# TT Project

Проект является тестовым для компании Tashkent university of Technology
Примитивный мессенджер с авторизацией через Google и GitHub (Supabase).

# Установка
1. Создаём .env файл с переменными
```copy
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
```
Ключи можно получить у разработчика

2. Устанавливаем зависимости
```bash
npm install
```

3. Запускаем для просмотра в тестовой среде
```bash
npm run dev
```

В проекте реализована доп команда /weather

```copy
/weather
```
